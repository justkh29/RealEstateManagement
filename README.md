# RealEstateManagement

A blockchain-based real estate management system built using **Vyper smart contracts**. It explores how real estate assets can be represented, managed, and interacted with on-chain, with supporting Python utilities for cryptography, IPFS storage, and a basic GUI.

## Features

- **Vyper smart contracts** for blockchain logic
- Real estate asset management concepts
- Account and crypto utilities
- IPFS integration for decentralized storage
- Python GUI for interacting with the system
